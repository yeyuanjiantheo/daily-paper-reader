## 摘要
本文检验了图神经网络能否改进已实现波动率的预测，以及这些预测能否提升投资组合表现。利用2015-2025年间标普500指数中465只股票的周度已实现波动率，将异质自回归模型和长短期记忆网络基线模型与基于滚动相关性、行业和格兰杰因果图构建的GraphSAGE模型进行对比，同时考虑宏观状态特征。实证发现，预测均方误差最低的模型、截面排序准确率最高的模型以及投资组合夏普比率最高的模型是三个不同的模型。预测精度、排序质量和投资组合表现是相关但不可互换的目标。只有当投资组合规则能够利用图模型所编码的截面结构时，图波动率模型才具有价值。

## Abstract
This paper tests whether graph neural networks improve realized volatility forecasts and whether those forecasts improve portfolio performance. Using weekly realized volatility for 465 S&P 500 equities from 2015-2025, Heterogeneous Autoregressive and Long Short-Term Memory baselines are compared against GraphSAGE models built on rolling correlation, sector, and Granger-causal graphs, with and without macro regime features. The empirical finding is that the model with the lowest forecast MSE, the model with the highest cross-sectional ranking accuracy, and the model with the highest portfolio Sharpe ratio are three different models. Forecast accuracy, ranking quality, and portfolio performance are related but not interchangeable objectives. Graph volatility models add value only when the portfolio rule can exploit the cross-sectional structure they encode.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：图神经网络（GNN）能否改进波动率预测？预测的提升是否能自动转化为更优的投资组合表现？
- **背景**：传统波动率模型（如HAR）捕捉单个股票的历史持续性问题，但忽视了波动在股票间的传染（如行业关联、相关性扩散）。GNN通过将股票视为节点、关系视为边，可以建模跨股票的信息传播，但图结构的有效性取决于边是否保持了样本外的信息。现有文献对“更好预测是否必然带来更好组合”缺乏系统验证。
- **含义**：论文揭示了一个重要的反直觉发现——预测均方误差（MSE）最低的模型、截面排序最准确的模型和组合夏普比率最高的模型是**三个不同模型**。这意味着预测精度、排序质量和投资组合表现是**相关但不可互换的目标**。图波动率模型只有在投资组合规则能利用其编码的截面结构时才产生价值。

## 2. 方法论

### 核心思想
- 使用**GraphSAGE**作为统一骨干网络，通过三种图构建策略（滚动相关、GICS行业、格兰杰因果）引入跨股票信息。对比经典线性模型（HAR）和深度学习序列模型（LSTM），并加入宏观状态特征（VIX、信用利差、市场平均相关性等）以分离“图结构”和“市场状态”的贡献。

### 关键技术细节
- **预测目标**：周度已实现波动率（年化日对数收益率标准差，每周计算，剔除少于3个交易日的周）。
- **特征组**：
  - 波动率特征：5/10/21/63天已实现波动率，短/长比。
  - 收益与交易量特征：5天、20天动量，对数成交量，成交量短/长比。
  - 宏观与状态特征：VIX及变化、SPY波动率和收益、10年期-2年期国债利差、投资级信用利差、平均配对相关性、图密度。
- **预处理**：个股特征每周截面winsorize（1%-99%）后z-score标准化；宏观特征用训练期统计量标准化，不进行截面缩放。
- **模型架构**：
  - **HAR**：两个变体——逐个股票独立回归（per-stock）和所有股票共用系数（pooled）。自回归使用5/21/63天波动率滞后。
  - **LSTM**：两层，隐藏维64，dropout 0.3，输入4周滚动窗口的10个个股特征，无跨股票信息。
  - **GNN骨干**：相同的GraphSAGE结构（均值聚合、线性变换），超参数通过验证集网格搜索（学习率1e-3，隐藏维256，3层，dropout 0.3，无归一化）。
- **三种图构建**：
  - **GNN-Correlation**：滚动Pearson相关系数绝对值≥0.3的股票对连无向边。每周围绕最新窗口更新，是动态图。考察不同窗口（21/63/126/252天）。
  - **GNN-Sector**：按照GICS行业分类连接同行业股票，每年更新一次，静态图。
  - **GNN-Granger**：基于训练期5天滞后的格兰杰因果检验（Bonferroni校正），建立有向边（A→B），静态图。
- **集成（Ensemble）**：三个GNN变体的预测按验证MSE逆加权平均。

## 3. 实验设计

- **数据集**：2015年1月–2025年12月标普500指数成分股（经95%价格覆盖率筛选后共465只股票）。注意：是当前成分股，存在幸存者偏差。
- **数据划分**：训练2015–2022，验证2023，测试2024–2025（共103周）。
- **基线方法**：
  - HAR per-stock（每个股票单独线性回归）
  - HAR pooled（所有股票共享系数）
  - LSTM（纯个股序列模型，无跨股票信息）
- **对比方法**：GNN-Correlation（多种窗口）、GNN-Sector、GNN-Granger、GNN-Ensemble，以及它们加入宏观特征后的版本。
- **评价体系**：
  - **预测精度**：MSE（主要）、MAE、R²、方向准确率（DA）。
  - **截面排序能力**：每周Spearman等级相关系数（Rank IC）及其信息比（ICIR），顶部五分之一命中率（Top-Q HR），配对准确率（Pair Acc.）。
  - **投资组合表现**：四种策略（最小方差、逆波动率加权、波动率目标、多空），每周再平衡，10 bps交易成本，年化收益、年化波动、夏普比率、最大回撤、平均换手率。等权组合为被动基准。
- **超参数选择**：GNN超参数（学习率、隐藏维、层数、dropout、相关阈值、窗口长度）通过2023验证集网格搜索确定。HAR和LSTM作为固定基准，不调参。

## 6. 主要结论与发现

1. **预测精度、排序质量和组合表现是不同目标**：GNN-Corr.+Macro 63d MSE最低（0.0298），GNN-Ensemble+Macro Rank IC最高（0.438），GNN-Sector+Macro最低方差夏普最高（0.984）。三者不一致。
2. **宏特征效果显著**：加入宏观特征（VIX、信用利差等）后所有GNN模型提升，尤其对弱图结构（行业、格兰杰）帮助很大（MSE从0.0337降至0.0314左右）。
3. **动态相关图优于静态图**：仅滚动相关图（无宏特征）MSE优于HAR/LSTM，行业和格兰杰图更差。
4. **图模型在最小方差组合中表现突出**：最低方差组合中，GNN-Sector+Macro和GNN-Granger+Macro夏普达0.98左右，远超HAR（0.64–0.73）。
5. **逆波动率加权无法击败等权**：所有模型逆波动率组合夏普均低于等权（0.513），最接近的是GNN-Sector+Macro 0.468。
6. **波动率靶向和多空策略均亏损**：波动率靶向所有模型夏普为负，多空策略全部负且严重，因2024–2025年高波动股票同时高收益。
7. **HAR依然是最强默认选择**：当简单、可解释和稳定性重要时，HAR足够好。
8. **总结性论断**：更好的波动率预测不自动等于更好的投资组合，改进的传递取决于组合构造对预测截面结构的利用程度。

## 7. 优点

- **问题设计新颖**：清晰区分“预测精度”和“投资组合价值”两个不同目标，并实证证明其不等价性。这打破了金融机器学习中常见的“更好的预测=更好的决策”隐含假设。
- **方法分解系统**：通过固定GraphSAGE骨干、变化图构建和宏特征有无，实现了对图结构来源和机制贡献的定位。
- **评价体系完整**：从统计精度（MSE等）、到排序能力（Rank IC）、再到多种组合策略，覆盖了从模型输出到实际应用的完整链条。
- **实验公平**：基线和GNN均在相同数据、相同训练/验证/测试划分下比较，并报告了多类指标而非单一指标。
- **实操指导意义**：结论如“HAR是默认选择”“图模型仅在组合规则能利用截面结构时有价值”为从业者提供了具体决策框架。

## 8. 不足与局限

- **幸存者偏差**：股票池为当前标普500成分股，剔除了期间退出或退市股票，使回测条件比真实更为理想化。
- **测试窗口短且特殊**：仅103周（2024–2025），该时期由科技和AI板块主导，高波动股票伴随高收益，导致多空策略全面失败。结论可能不能推广至其他市场状态（如危机期或价值回归期）。
- **格兰杰图静态**：整个训练期仅估计一次，未能适应市场结构变化。虽然滚动相关图动态更新，但窗口长度固定，可能滞后于快速变化的关系。
- **交易成本简化**：假设统一10 bps且每周再平衡，未考虑市场冲击、流动性限制、约束变化等实际摩擦。
- **组合策略有限**：仅测试四种简单构造，未考虑均值-方差优化、风险平价、Black-Litterman等更复杂的组合方法。
- **未控制过平滑**：在危机期图密度极高（如COVID时平均邻居432.7/464），GNN可能过度平滑预测，论文仅提及但未做技术处理。
- **局限性明确列出**：作者在“7.1 Limitations”中已诚实承认上述多数问题，并给出未来方向（自适应图、过平滑控制、组合感知训练损失等）。
