## 摘要
本文检验了图神经网络是否改进了已实现波动率的预测，以及这些预测是否提升了投资组合的表现。使用2015-2025年间465只标普500股票的周度已实现波动率，将异质自回归和长短期记忆基线模型与基于滚动相关性、行业和格兰杰因果图构建的GraphSAGE模型进行了比较，其中部分模型纳入了宏观状态特征。实证发现：预测均方误差最低的模型、横截面排序准确率最高的模型以及投资组合夏普比率最高的模型是三个不同的模型。预测准确性、排序质量和投资组合表现之间存在关联，但并非可互换的目标。只有当投资组合规则能够利用图波动率模型所编码的横截面结构时，图波动率模型才具有价值。

## Abstract
This paper tests whether graph neural networks improve realized volatility forecasts and whether those forecasts improve portfolio performance. Using weekly realized volatility for 465 S\&P 500 equities from 2015--2025, Heterogeneous Autoregressive and Long Short-Term Memory baselines are compared against GraphSAGE models built on rolling correlation, sector, and Granger-causal graphs, with and without macro regime features. The empirical finding is that the model with the lowest forecast MSE, the model with the highest cross-sectional ranking accuracy, and the model with the highest portfolio Sharpe ratio are three different models. Forecast accuracy, ranking quality, and portfolio performance are related but not interchangeable objectives. Graph volatility models add value only when the portfolio rule can exploit the cross-sectional structure they encode.

- **研究动机**：波动率预测是资产配置的核心，但预测精度的提升是否必然转化为投资组合性能的提升仍不明确。传统模型（如HAR、LSTM）主要捕捉单个股票自身的历史波动率，但忽略了波动率在股票之间的跨市场传播。图神经网络能够建模股票间的相互关系，但图结构的有效性取决于边缘是否在样本外保持信息量。
- **核心问题**：
  1. GNN是否能够比强基线模型（HAR、LSTM）更准确地预测周度已实现波动率？
  2. 这种预测改进是否能够转化为更好的投资组合表现？
- **整体含义**：预测精度、排序质量和组合表现是相关但不可互换的目标；图神经网络仅在投资组合规则能够利用其编码的横截面结构时才有价值。

## 论文提出的方法论

- **核心思想**：通过图神经网络建模股票间的跨截面依赖关系，将波动率信号在关联股票间传播，以捕捉单一模型无法直接捕获的波动率传染效应。
- **模型架构**：
  - **HAR（异质自回归模型，Corsi 2009）**：线性基线，使用5、21、63个交易日的平均已实现波动率预测下一周波动率。包括每个股票独立拟合（per-stock）和所有股票共同拟合（pooled）两种变体。

  - **GNN（图神经网络）**：基于GraphSAGE骨架，每层进行邻居特征平均、与自身特征拼接、线性变换。所有图变体共享相同超参数（学习率1e-3、隐藏维度256、3层、dropout 0.3）。

## 论文的主要结论与发现

1. **预测精度与组合表现不是同一目标**：最低MSE的模型（GNN-Correlation+Macro 63d，MSE=0.0298）、最高Rank IC的模型（GNN-Ensemble+Macro，Mean IC=0.438）和最高最小方差夏普比率的模型（GNN-Sector+Macro，Sharpe=0.984）各不相同。
2. **动态相关图优于静态图**：GNN-Correlation（动态）优于GNN-Sector（静态）和GNN-Granger（静态）。但宏观特征能弥补静态图的不足（GNN-Sector+Macro MSE从0.0336降至0.0315）。
3. **宏观特征关键**：几乎所有GNN变体加入宏观特征后MSE和组合表现均改善。宏观特征提供市场状态感知，与图结构互补。
4. **组合规则影响结果**：
   - 最小方差组合：图模型+宏观特征大幅超越基线（HAR per-stock Sharpe 0.635 vs. GNN-Sector+Macro 0.984）。
   - 逆波动率加权：所有模型均未跑赢等权重组合（等权重Sharpe 0.513，最佳模型GNN-Sector+Macro 0.468）。
   - 波动率目标：所有模型均为负夏普比率。
   - 多空组合：全负，因测试期高波动（科技/AI）股票同时有高收益。
5. **HAR仍为强基线**：HAR per-stock在MSE（0.0329）和夏普比率（最小方差0.635）上表现稳健，复杂度低，适合偏好简单性的场景。
6. **应用建议**：GNN适合大市值、市场关系有信息、有宏观特征、组合规则能利用截面结构的场景；否则HAR是更好的默认选择。

1. **动态格兰杰因果图**：当前 GNN‑Granger 仅基于训练期一次估算，未来可采用滚动窗口重新估算因果边，使图结构适应市场状态变化，可能进一步提升预测能力。
